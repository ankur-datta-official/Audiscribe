Technologies Used

1. Python

Python is the core language used for backend development, especially for handling speech recognition, machine learning, and processing data using its powerful libraries like speech_recognition, pydub, and moviepy.

2. Flask

Flask is a lightweight Python web framework used to serve the frontend pages and link the backend processing with user interactions. It handles routes, file uploads, and dynamic rendering with Jinja2 templating.

3. TensorFlow / Keras

Used to load and deploy the pre-trained deep learning model asl_model.h5 for classifying American Sign Language (ASL) gestures from video input. These frameworks enable real-time image-based classification using Convolutional Neural Networks (CNN).

4. HTML, CSS & JavaScript

The user interface is built using standard web technologies. HTML structures the content, CSS styles the layout and appearance, and JavaScript enhances interactivity (e.g., file input behavior, theme toggling).

5. Speech Recognition Library

This Python library converts audio (e.g., WAV or MP3) to text. It uses Google’s Web Speech API to interpret audio content, which is then rendered on the webpage and saved as a downloadable .txt file.

6. MoviePy & PyDub

MoviePy is used to extract audio from video files, and PyDub processes and converts the audio into suitable formats for recognition. These tools are essential for handling multimedia data.

7. OpenCV

OpenCV is optionally used for handling image and video frame extraction during sign language recognition. It helps process video streams and isolate frames for model inference.

8. Bootstrap (Optional)

If added, Bootstrap enhances UI components with responsive design and prebuilt classes for buttons, forms, and layout alignment.

9. Google Fonts

The “Inter” font family is imported for a clean and modern look across all frontend pages using Google Fonts CDN.
